# Talha Rauf — Developer Portfolio

Personal portfolio website built with plain HTML, CSS, and vanilla JavaScript. Deployed on GitHub Pages.

🌐 **Live Site:** [talha-rauf-dev.github.io](https://talha-rauf-dev.github.io)

## About

This is my personal developer portfolio showcasing my projects, skills, and work experience as a Sr. Full-Stack Engineer. It covers backend systems, cloud infrastructure, and full-stack development work across CRM, ERP, and real-time application domains.

## Built With

- HTML5
- CSS3 (custom, no frameworks)
- Vanilla JavaScript
- EmailJS (contact form)
- Font Awesome (icons)
- Google Fonts — Inter

## Sections

- **Hero / About** — intro and tagline
- **Projects** — work across CRM platforms, EV tech, AI content, and more
- **Skills** — tech stack grouped by category
- **Experience** — work history across GCS Pvt. Ltd., SolutionsLoft, and Intellisence
- **Contact** — EmailJS-powered contact form with direct contact details

## Deployment

This site is deployed via GitHub Pages directly from the `main` branch with no build step required.
